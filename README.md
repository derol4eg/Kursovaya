## ⚙️ Инструкция по запуску

### 1. Подготовка окружения
Убедитесь, что у вас установлен Docker Desktop и запущен PowerShell от имени администратора.

### 2. Развертывание контейнеров
Запустите кластер (выполняется один раз или при сбросе данных):

docker-compose down
docker-compose up -d

### 3. Запуск самого скрипта

.\PUSK.ps1
